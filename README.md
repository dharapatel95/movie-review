# IMDB Movie Review Sentiment Analysis using RNN

A deep learning web application that classifies IMDB movie reviews as **Positive** or **Negative** using a Simple Recurrent Neural Network (RNN).

## Overview

This project uses the IMDB movie review dataset from TensorFlow/Keras. The model learns patterns from review text and predicts the sentiment of a user-entered movie review.

## Features

- Classifies movie reviews as Positive or Negative
- Uses a Simple RNN model for sentiment analysis
- Text preprocessing using word indexing and sequence padding
- Interactive web interface built with Streamlit
- Displays prediction score for each review

## Tech Stack

- Python
- TensorFlow / Keras
- NumPy
- Streamlit
- Natural Language Processing (NLP)
